# 🪵 Tung Tung Sahur Wood Theme

An earthy, woody, and ultra-aesthetic VS Code theme inspired by the viral 2025 internet meme character **Tung Tung Tung Sahur**. Bring the warm, sepia-toned vibes of an anthropomorphic Kentongan straight to your favorite code editor!

Whether you are coding late into the night or waking up early for Sahur, this theme provides the perfect high-contrast, low-strain environment for your eyes.

## ✨ Features

* **Tung Tung Sahur Wood (Dark):** A deep, nocturnal palette featuring rich dark browns, warm burnt oranges, and soft sepia accents. Reminiscent of a glowing streetlamp in the quiet hours of the night.
* **Tung Tung Sahur Wood (Light):** A crisp, daytime variant utilizing bamboo tones, parchment backgrounds, and terracotta syntax highlighting for a natural, grounded coding experience.
* **Optimized Syntax Highlighting:** Carefully tuned colors for maximum readability across popular languages (Java, C/C++, Python, HTML/CSS, etc.).
* **Skeuomorphic & Earthy:** Ditch the neon cyberpunk vibes for pure, unadulterated *wood*.

## 🚀 Installation

1. Open **Extensions** sidebar panel in Visual Studio Code. `View → Extensions`
2. Search for `Tung Tung Sahur Wood`
3. Click **Install**
4. Click **Reload** to reload your editor.
5. Navigate to `Settings > Theme > Color Theme` and select **Tung Tung Sahur**.


## 📖 About the Lore

For the uninitiated, **Tung Tung Tung Sahur** (/ˈtuːŋ ˌsaˈhur/ *TOONG sa-HOOR*) is a viral Internet meme character introduced in 2025 by TikTok user `@noxaasht` as a part of the *Italian brainrot* series. Despite the series' name, it is of Indonesian origin.

The character is an AI-generated anthropomorphic Kentongan (a traditional Indonesian slit drum) holding a wooden drumstick (sometimes identified as a baseball bat). It has since been featured in many songs and video games, most notably *Steal a Brainrot*.

Now, he is here to oversee your code and make sure you never miss a semicolon.
